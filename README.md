# E-Commerce Sales & Customer Analytics

## Overview

This project analyzes over 1 million e-commerce transactions to understand sales performance, customer behavior, and revenue trends. The goal is to transform raw transactional data into meaningful business insights and create a portfolio-ready analytics project.

## Objective

The objective of this analysis is to:

- Understand overall sales and revenue performance
- Identify customer purchasing behavior patterns
- Detect data quality issues such as missing values and returns
- Create a structured dataset for analysis and dashboarding
- Present findings in a clear, business-focused format

## Tools Used

- Python (Pandas, NumPy)
- SQL
- Excel
- Tableau
- Jupyter Notebook

## Dataset

The dataset contains e-commerce transaction records with information such as transaction ID, product details, quantity, transaction date, unit price, customer ID, and country.

## Data Preparation

The dataset was explored and prepared through the following steps:

- Loaded raw transactional data into Python
- Reviewed dataset structure, data types, and summary statistics
- Identified missing values and duplicate records
- Converted date fields to datetime format for time-based analysis
- Detected return transactions using negative quantities
- Created a revenue metric using Quantity × Price
- Prepared the data for further analysis and visualization

## Analysis Performed

The project includes analysis of:

- Total revenue
- Transaction volume
- Customer activity
- Product performance
- Return behavior
- Time-based sales trends

## Key Insights

- The dataset contains over 1 million transactions, indicating high-volume retail activity
- Return transactions are present and should be considered in analysis
- Customer identifiers contain missing values that affect customer-level analysis
- Revenue was successfully engineered as a core business metric
- The dataset is suitable for deeper analysis such as segmentation, trend analysis, and dashboard creation

## Deliverables

- Cleaned and explored dataset
- Python notebook with data preparation and analysis
- Excel-based analysis and supporting calculations
- Tableau dashboard for visual storytelling
- GitHub repository documenting the full project

## Outcome

This project demonstrates end-to-end data analysis skills including data cleaning, transformation, exploration, and business insight generation. It reflects the type of analytical work expected in entry-level data analyst roles.
